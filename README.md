# 📌 Planora - Smart Productivity & Task Management System

Planora is a JavaFX-based desktop application designed to help users organize tasks, manage time efficiently, and collaborate with teams — all in one place.

---

## 🚀 Features

### 📝 Task Management

* Create and save tasks easily
* Tasks can be filtered based on:

    * 📅 Today
    * 📆 Upcoming
* Tasks can be categorized into:

    * Study
    * Personal
    * Health
    * Work
    * Others

---

### ✅ Task Tracking

* Mark tasks as completed
* Maintain history/records of completed tasks

---

### 📅 Calendar Integration

* Built-in calendar system
* View tasks based on selected dates
* Easily track scheduled activities

---

### 🎧 Focus Mode

* Dedicated focus environment with timer
* Includes 3 relaxing modes:

    * 🌧️ Rain Mode
    * 🌙 Night Mode
    * 🌿 Nature Mode

---

### 👥 Space (Team Collaboration)

* Create teams/groups (Spaces)
* Invite members to collaborate
* Assign and manage tasks within groups
* 💬 Real-time chat system inside each space

---

### 🔐 Authentication System

* User login system
* Forgot password functionality
* Password change support

---

## 🛠️ Technologies Used

* Java (JDK 21)
* JavaFX
* Maven
* MongoDB Database
* ControlsFX & FXGL libraries

---

## 📦 How to Run the Application

### 🔹 Option 1: Run Executable (Recommended)

1. Open the folder:

   ```
   Planora-App/
   ```
2. Double-click:

   ```
   Planora.exe
   ```

👉 This will launch the application directly.

---

### 🔹 Option 2: Run from Source Code

#### Prerequisites:

* Java JDK 21+
* Maven installed

#### Steps:

1. Clone the repository:

   ```
   git clone <your-repo-link>
   ```

2. Navigate to the project:

   ```
   cd Planora
   ```

3. Run the project:

   ```
   mvn clean javafx:run
   ```

👉 OR run manually:

```
Run Launcher.java
```
